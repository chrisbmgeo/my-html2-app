# Η Κατάρα του Τουταγχαμών

Μια διαδραστική ιστοσελίδα που εξερευνά τον μύθο της "Κατάρας του Τουταγχαμών", αναλύοντας τα ιστορικά γεγονότα και καταρρίπτοντας τους μύθους.

## Περιγραφή
Η εφαρμογή παρουσιάζει μια αφήγηση σε 7 βήματα, καλύπτοντας την ανακάλυψη του τάφου, τον θάνατο του Λόρδου Κάρναρβον, και τα στατιστικά στοιχεία που αποδεικνύουν την έλλειψη "κατάρας".

## Πώς να το ανοίξετε
Απλά ανοίξτε το αρχείο `index.html` στον browser σας (Chrome, Firefox, Edge).

## Τεχνολογίες
- HTML5
- Tailwind CSS (μέσω CDN)
- Vanilla JavaScript
